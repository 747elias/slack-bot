This is my selfmade Slack bot. He has really basic functions.

## What he can do:
- `/elias-ping` - Check bot latency
- `/elias-catfact` - Get a cat fact
- `/elias-joke` - Get a nice joke
- `/elias-duck` - Get a random duck image
- `/elias-advice` - Get a life advice
- `/elias-qr <text>` - Get a QR Code

## How to install:
1. Download the Github repo.
2. Make sure you have Node.js installed.
3. Add a `.env` file and fill in your `SLACK_BOT_TOKEN` and your `SLACK_APP_TOKEN`.
4. Go to the folder in the terminal and run `node index.js`.
5. Enjoy!
